Come installare Open Mod Manager per DCS

Guida in italiano per installare Open Mod Manager, configurare i canali DCS, aggiungere repository e gestire mod scaricate online o manualmente.

Prima di iniziare

Open Mod Manager, abbreviato OMM, permette di installare e rimuovere mod senza copiare file a mano dentro DCS. Per una gestione ordinata è consigliato creare prima una cartella chiamata DCS Mods, anche su un disco esterno, dove conservare mod, liveries e pacchetti della squadriglia.

Questa pagina traduce e adatta la procedura operativa fornita, senza mantenere la numerazione originale del documento sorgente.

Procedura

Segui gli step nell’ordine. L’idea di base è creare un HUB, configurare due canali principali — OpenBeta e Saved Games — e poi usare repository o libreria locale per installare le mod.

1

Scarica e installa Open Mod Manager

Apri la pagina di download di Open Mod Manager e scarica l’ultima versione in formato eseguibile, scegliendo x64 o x86 in base al tuo sistema operativo. Avvia il file scaricato e completa il wizard di installazione.

Per evitare problemi di permessi, è consigliato avviare OMM come amministratore, specialmente quando deve scrivere nella cartella principale di DCS.

2

Crea il Mod HUB

Al primo avvio vai su FILE > NEW > MOD HUB.... Nel campo HUB NAME inserisci un nome chiaro, ad esempio DCS Mods Hub, così sarà facile distinguerlo da eventuali altri hub.

Seleziona una cartella dove OMM creerà e salverà i file necessari. Puoi crearla vicino alla cartella DCS Mods preparata in precedenza. Quando compare il popup Adding Channel, seleziona YES.

3

Configura il canale Main OpenBeta

Nel wizard del nuovo canale inserisci come Channel title il nome DCS Open Beta. Nel campo successivo indica il percorso della cartella principale di DCS, di solito:

C:\Users\TUO_UTENTE\Saved Games\DCS.openbeta

Nella sezione seguente puoi abilitare Use custom directory for Mods Library per indicare una cartella diversa di salvataggio delle mods da OMM come DCS Mods, altrimenti crea una cartella Library dentro la cartella DCS_Open_Beta sul DCS Mods Hub.

La parte backup può essere saltata se non richiesta dalla squadriglia. Premi Finish.

4

Aggiungi una repository

Le repository sono archivi online da cui OMM può leggere, scaricare e aggiornare pacchetti mod senza dover copiare manualmente i file.

Per aggiungerne una vai su CHANNEL > ADD REPOSITORY..., oppure apri la scheda Network Library e premi il pulsante + verde accanto alla lista repository. Fai attenzione a non premere il + della lista canali.

Inserisci URL e nome della repository. Se i dati sono corretti, OMM mostrerà il messaggio Repository appear valid. Premi Finish.

NOTA: I nomi corretti e i link delle repository saranno disponibili negli appositi canali Discord una volta effettuata l’iscrizione alla squadriglia.

5

Scarica e installa un pacchetto dalla repository

Nella schermata repository fai doppio click sulla repository da interrogare, oppure selezionala e usa la freccia blu laterale. Comparirà la lista dei pacchetti scaricabili. Se interroghi più repository, OMM può mostrare i pacchetti disponibili in un’unica lista.

Seleziona il pacchetto desiderato e premi Download in basso a sinistra. Segui l’avanzamento dalla barra di completamento. Una volta terminato, torna alla scheda Local Library. Se il pacchetto non compare subito, riavvia OMM per aggiornare la lista.

Con il pacchetto selezionato premi Install. Per rimuoverlo, premi Uninstall.

6

Aggiungi una mod scaricata manualmente

Se scarichi una mod fuori da una repository, salvala nella cartella Library usata come libreria locale. OMM legge automaticamente cartelle, archivi .zip e pacchetti .ozp, purché siano strutturati correttamente.

Altrimenti seleziona l'icona a destra a forma di pacco con un piccolo + bianco dentro un quadrato verde, seleziona il pacchetto .zip desiderato e premi apri. Seleziona il pacchetto, scegli il canale corretto e premi Install.

Quando vuoi rimuoverla, seleziona la stessa mod e premi Uninstall. OMM eliminerà i file installati e ripristinerà la configurazione precedente, se il pacchetto è stato creato correttamente.

Checklist FD

Prima della missione

  • OMM avviato come amministratore
  • Cartella DCS Mods creata
  • Canale Main OpenBeta configurato
  • Canale Saved Games configurato
  • Repository corrette aggiunte
  • Mod installate nel canale giusto
  • Mod non richieste disattivate
Regola pratica

Se non sei sicuro del canale, chiedi prima sul Discord FD: installare una mod nella cartella sbagliata può impedire l’ingresso al server o creare conflitti.
